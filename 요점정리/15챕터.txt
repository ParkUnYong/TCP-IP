fgetc,fputs,fopen,feof


fgetc(FILE *stream) 
키보드 대상으로 문자 입력 받음.(stdin) 
파일에 저장된 문자 하나를 반환함. 해줌 (파일로부터 입력받음.)
파일의 끝에 도달하거나 호출 실패시 EOF 반환

EOF는 -1을 의미함 

fputs(const char *s,FILE *STREAM)
스트림으로 하나의 문자열을 출력할때 사용 
성공시 음수가 아닌 값을 실패시 EOF반환
stdout 같이 모니터 스트림을 넣으면 입력 받은 문자열을 출력하고
파일에다가 하면 그 파일의 값을 출력함. 

puts(const char *s)
puts는 스트림을 지정 못함. 모니터로 고정 

puts함수는 문자열 출력후 자동으로 개행이 나오고 fputs는 그게 아니라서 개행을 해줘야함.

feof(FILE *stream)
인자로 넘겨준 스트림의 파일 끝 표시자를 검사하여 설정(set)되어 있다면 0이 아닌 값을 리턴 
실패하면 0이 리턴.

fopen(const char *filename,const char *mode)
성공시 파일의 FILE구조체 변수의 주소값, 실패시 NULL 포인터 반환

첫번째 인자는 파일이름, 두번쨰 인자는 옵션임
rt 읽기 : 파일에서 데이터를 입력버퍼로 읽어옴.
wt 쓰기 : 프로세스에서 데이터를 출력버퍼로 보내 파일에 저장


이러한 방식을 버퍼링
